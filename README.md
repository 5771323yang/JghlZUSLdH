# 校园管理系统 java692

## 项目概述

校园管理系统是基于Spring Boot和Vue技术栈开发的应用，实现了对校园内用户、院校以及通知推送的高效管理。

## 技术栈

- 开发环境：Windows 10
- 架构模式：MVC（Model-View-Controller）/前后端不分离
- JDK版本：Java JDK 1.8
- 开发工具：IDEA
- 数据库版本：MySQL 8.0
- 数据库可视化工具：Navicat
- 服务器：Spring Boot 自带 Apache Tomcat
- 主要技术：Java、Spring Boot、MyBatis、MySQL、jQuery、HTML、Vue

## 功能模块

### 系统角色

- 用户
- 管理员
- 院校管理员

### 管理员功能

- 首页
- 个人中心
- 院校管理
- 用户管理
- 单位类别管理
- 院校管理员管理
- 单位管理
- 通知推送管理
- 投票信息管理
- 通知回复管理

### 用户功能

- 首页
- 个人中心
- 单位管理
- 通知推送管理
- 投票信息管理
- 通知回复管理

### 院校管理员功能

- 首页
- 个人中心
- 用户管理
- 单位类别管理
- 单位管理
- 通知推送管理
- 投票信息管理
- 通知回复管理

## 运行环境

- 服务器：Apache Tomcat（Spring Boot 内嵌）
- 数据库：MySQL 8.0
- 前端：Vue

## 部署步骤

1. 确保MySQL服务已启动且可连接。
2. 导入数据库结构及初始数据。
3. 编译项目并打包为可执行的JAR文件。
4. 在服务器上运行打包好的JAR文件。
